# 内置 image_gen 提示词模板

当内置 `image_gen` 不能接收真正的遮罩参数时，用这份模板把灰黑分区遮罩图作为参考图传达给模型。

## 基础模板

```text
请把原图作为编辑目标图。
请把遮罩图只当作“哪些区域允许修改，以及哪些区域只能联动光影”的参考图。

遮罩语义：
- 遮罩中的灰色区域是联动区，结构和文字必须保持不变，只允许发生轻微的亮度、对比度、色温和阴影色偏联动。
- 遮罩中的黑色区域是可编辑区，允许修改。
- 灰色区域通常由若干连续区域组成，请把每个连续灰区理解为单独的联动表面，不要把整片建筑统一重绘。

任务：
只修改黑色可编辑区，把它变成：<具体改动目标>。
灰色联动区只能适应新的环境光，不允许改结构和内容。

保护要求：
- 不要重绘、移动、重构、重塑或重新设计灰色联动区。
- 灰色联动区中的人物、建筑、文字、logo、边缘、透视和纹理都要尽量保持和原图一致。
- 灰色联动区只允许非常轻微的环境光变化，例如亮度、对比度、阴影色偏和暖冷关系调整。
- 保持原始构图和拍摄角度不变。
- 不要在可编辑区之外新增物体。
- 灰色联动区边界附近要过渡自然，避免光晕、模糊、涂抹和串色。

输出要求：
- 成图应当看起来像原图只修改了可编辑区，同时灰色区只发生受控的光影联动。
- 优先最小化干预。
- 除非明确要求，否则不要做全局改色或整体风格重做。
```

## 无遮罩直改天空模板

当用户明确不想画遮罩，但仍要求“只改天空和整体色调，不改建筑等固定内容”时，用下面这份更强约束模板。

```text
请把原图作为编辑目标图。

任务：
- 只修改天空本身的云层、明暗、色彩和天空氛围。
- 同时只允许做非常轻微、真实的整体色调统一，让地面景物自然吃到新的环境光。

严格限制：
- 不要修改任何固定主体的结构、形状、位置、比例、透视和边缘。
- 不要重绘、移动、增删或重塑建筑、人物、文字、logo、树木、水岸、山体、栏杆、道路、灯具、船只和其他实景物体。
- 不要改变建筑细节、屋檐轮廓、塔刹、窗格、砖石纹理、岸线走向、水面边界和远山轮廓。
- 不要新增任何新的物体、装饰、灯光、飞鸟、云隙造景、倒影元素或幻想效果。
- 除天空之外，其余区域应尽量保留原图像素观感，只允许出现非常轻微的曝光、对比度和色温联动。
- 如果某个改动会导致固定主体被重绘，那么宁可少改，也不要去改主体。

风格要求：
- 目标是更好看的真实照片，不是重生成一张新图。
- 天空可以从沉闷阴天调整为更通透、更有层次、但仍然自然可信的傍晚或晚霞天空。
- 整体色调只做克制的暖色环境光统一，避免重度 HDR、影视化过强、奇异橙紫色、赛博色和插画感。
- 保持原始构图、拍摄角度、焦段感和照片真实性。

输出要求：
- 成图应看起来像同一张照片只换了天空并做了轻微整体调色。
- 优先最小化编辑范围。
- 建筑和其他固定物体必须尽量接近原图。
```

## 二次收紧补充块

当第一轮结果不稳时，只追加一小段，不要整段重写。

### 收紧保护区

```text
上一版在灰色联动区之外有改动。请进一步收紧保护范围，让灰色区在纹理、形状和边缘细节上更接近原图，只保留很轻微的环境光变化。
```

### 收紧边界

```text
请让可编辑区和灰色联动区的交界更干净、更自然，避免边界发灰、发糊或出现溢出。
```

### 收紧整体氛围漂移

```text
不要重塑整张图的风格。只改可编辑区，并让灰色区只做轻微光影联动，整体氛围仍然接近原图。
```

### 收紧无遮罩主体漂移

```text
上一版对固定主体改动过大。请进一步收紧：除了天空和非常轻微的整体色调统一之外，不要改动任何建筑、树木、水岸、远山和水面结构；如果无法同时满足，就优先保留原图主体不变。
```
